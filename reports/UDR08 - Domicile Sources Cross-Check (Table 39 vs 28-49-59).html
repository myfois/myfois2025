UDR08 — UEL Domicile Sources Cross-Check

Independently verifying UEL enrolment figures across four separate HESA tables (28, 39, 49, 59), 2016/17–2018/19 – the only years all four cover in this form. University of East London (UKPRN 10007144).

SCOPE Table 39 ("HE student enrolments by provider, domicile, level, mode") only exists for 2016/17–2018/19 – HESA discontinued it after that. That's why this cross-check is limited to those 3 years, unlike the Student Origin Explorer's full 2014/15–2024/25 range built from Tables 28 and 59.

Reading this

All three sources agree, everywhere, within HESA's rounding tolerance. Across all 63 level × mode × year combinations, Table 49's total, Table 39's own total, and the sum of Table 28 (non-UK) + Table 59 (UK) never differ by more than ±5 – exactly the size of HESA's disclosure-control rounding, and no more. This is a genuine independent cross-check across four separately-published tables, not a single source checked against itself.
LIMITATION Because Table 39 stops at 2018/19, this cross-check can't validate the more recent, much larger international growth documented in the Student Origin Explorer (built from Tables 28/59, which run to 2024/25). It confirms the method is sound for the years it covers, not that later years are correct by extension.
